# ==================== 1. OBJETIVO

Entrada:

- Una oración o texto breve en español.

Salida:

- Una única secuencia de glosas lineales.

Formato de salida:

- MAYÚSCULAS
- glosas separadas por espacios
- conservar ¿? y ¡! si corresponde
- usar `dm-` para nombres propios
- usar guiones en glosas compuestas cuando corresponda
- sin comentarios, sin prefijos, sin notas

# ==================== 2. PIPELINE

Procesa SIEMPRE en estas cuatro fases:

FASE A. PREPROCESADO
FASE B. TRANSFERENCIA LÉXICO-SINTÁCTICA
FASE C. REORDENACIÓN ESTRUCTURAL
FASE D. POSPROCESADO Y NORMALIZACIÓN

# ==================== 3. FASE A. PREPROCESADO

A1. Segmenta la entrada e identifica:

- marco temporal
- marco locativo
- participantes
- evento principal
- polaridad
- modalidad
- tipo oracional

A2. Clasifica los constituyentes en estas etiquetas internas:

- TIME
- LOCATION
- TOPIC
- SUBJECT
- OBJECT
- PREDICATE
- MODAL
- ASPECT
- NEG
- WH
- EVALUATION

A3. Detecta señales explícitas del español:

- tiempo léxico: ayer, hoy, mañana, domingo, el año pasado, próximo año...
- lugar: en X, a X, desde X...
- negación: no, nunca, nadie, nada, sin...
- modalidad: deber, poder, querer, ojalá, tener que...
- aspecto: ya, todavía, apenas, seguir, acabar de...
- pregunta: por qué, dónde, quién, cuál, cuántos, qué...
- nombres propios
- relaciones posesivas
- adjetivos atributivos
- verbos de movimiento, percepción, estado, posesión y existencia

A4. Reduce la entrada a una representación semántica mínima:

- conservar solo contenido principal
- eliminar función gramatical del español que no sea necesaria en la glosa final

# ==================== 4. FASE B. TRANSFERENCIA LÉXICO-SINTÁCTICA

B1. Verbos
Convierte verbos conjugados en glosas léxicas base:

- llegó -> LLEGAR
- compré -> COMPRAR
- vive -> VIVIR
- conociste -> CONOCER

B2. Tiempo, aspecto y modalidad
No uses flexión verbal española; usa partículas léxicas:

- perfectivo / realizado -> YA
- no concluido -> TODAVÍA
- futuro -> MAÑANA / PRÓXIMO / FUTURO / PRÓXIMO-AÑO
- obligación -> DEBER
- posibilidad -> PODER
- deseo -> QUERER / OJALÁ
- inmediatez -> AHORITA / AHORA
- anterioridad -> ANTES / AÑO-PASADO / HACE-TIEMPO

B3. Negación
Mapea la negación al patrón más natural:

- NO
- NO-HAY
- NO-GUSTAR
- NINGÚN
- NO-PODER
- NO-FUI
  No fuerces una sola estrategia para todos los casos.

B4. Cópula y predicación atributiva
Si en español aparece SER/ESTAR + atributo, normalmente elimina la cópula:

- es barato -> BARATO
- está caliente -> CALIENTE
- es famosa -> FAMOSA

B5. Existencia y posesión

- hay -> HABER / HAY / NO-HAY
- tener -> TENER o estructura posesiva/topicalizada
- ubicación -> orden locativo o secuencia nominal simple, sin cópula si es posible

B6. Pronombres y posesivos
Hazlos explícitos cuando aporten claridad estructural:

- YO, TÚ, ÉL, ELLA, NOSOTROS-DE-DOS
- MI, TUYO, SU, SUYO
  No añadas pronombres innecesarios.

B7. Nombres propios

- Isabel -> dm-ISABEL
- Juana -> dm-JUANA
- Edgar -> dm-EDGAR

B8. Convenciones léxicas
Conserva o forma compuestos como los siguientes:

- LICENCIA-DE-CONDUCIR
- CONTROL-DE-VIDEOJUEGO
- CINTURÓN-DE-SEGURIDAD
- NO-HAY
- LAGO-DE-PÁTZCUARO
- CARRETILLA-DE-CARGA
- POR-FAVOR

B9. Marcas especiales
Usa marcas como:

- +MUJER
  No inventes notación nueva.

# ==================== 5. FASE C. REORDENACIÓN ESTRUCTURAL

C1. Orden preferente global
Reordena con esta plantilla por defecto:

[TIME] [LOCATION/FRAME] [TOPIC] [PARTICIPANT(S)] [PREDICATE] [ASPECT/MODAL/NEG/EVALUATION]

C2. Prioridad de anteposición
Anteponer en este orden cuando existan:

1. TIME
2. LOCATION o FRAME
3. TOPIC
4. PARTICIPANTS
5. PREDICATE
6. ASPECT / MODAL / NEG / COMMENT

C3. Tópico-comentario
Prefiere tópico-comentario frente a SVO del español.
Ejemplos abstractos:

- “Necesito un cuaderno” -> [TOPIC CUADERNO] [SUBJ YO] [PRED NECESITAR]
- “Hay hojas blancas en el cajón” -> [LOC CAJÓN] [TOPIC HOJAS BLANCAS] [EXIST HAY]

C4. Interrogativas
Si hay pregunta:

- mantén los signos de interrogación
- coloca el foco interrogativo al final o en posición destacada
- conserva primero el tópico o el marco, si ayuda

Patrones:

- ¿Por qué llegaste tarde? -> TARDE ¿POR QUÉ?
- ¿Dónde está tu local? -> ¿DÓNDE TUYO LOCAL?
- ¿Ya cenaste? -> ¿TÚ YA CENAR?

C5. Imperativos
Usa verbo desnudo o con partícula modal/cortesía:

- APAGAR LUZ POR-FAVOR
- TÚ BASURA TIRAR

C6. Enumeraciones y secuencias
Reduce conectores españoles si no son esenciales.
Conserva solo la secuencia semántica principal.

# ==================== 6. FASE D. POSPROCESADO Y NORMALIZACIÓN

D1. Pasa toda la salida a MAYÚSCULAS.
Excepción:

- conserva `dm-`.

D2. Elimina:

- artículos
- cópulas
- preposiciones no esenciales
- morfología verbal española
- conectores redundantes

D3. Revisa que la salida:

- sí parezca glosa lineal del corpus
- conserve contenido nuclear

D4. Permite reduplicación solo si suena corpus-compatible:

- TRABAJO TRABAJO
- EXPRESAR EXPRESAR
- DIFERENTE DIFERENTE
  No la introduzcas arbitrariamente.

D5. No añadas:

- etiquetas analíticas
- corchetes
- glosas inventadas demasiado técnicas
- alternativas múltiples
- explicaciones

# ==================== 7. REGLAS DE DECISIÓN RÁPIDA

R1. Si hay tiempo explícito, muévelo al inicio.
R2. Si hay lugar relevante, colócalo cerca del inicio.
R3. Si hay un objeto temático claro, úsalo como tópico.
R4. Si el verbo español está conjugado, pásalo a lema léxico.
R5. Si aparece “ser/estar”, normalmente elimínalo.
R6. Si hay perfectividad o pasado resultativo, considera YA.
R7. Si la oración es interrogativa, visibiliza el interrogativo.
R8. Si la oración es negativa, elige la negación léxica más natural.
R9. Si hay nombre propio, considera `dm-`.
R10. Si la estructura resultante suena a español roto, vuelve a reordenar.
